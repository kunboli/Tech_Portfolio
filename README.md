# Kunbo Li — IT Portfolio

A personal portfolio website showcasing technical projects across 
cybersecurity, artificial intelligence, agile delivery, and cloud 
engineering.

## Live Site
https://kunboli.github.io/Tech_Portfolio

## About
This portfolio was built as a single-page HTML/CSS/JS application 
with no frameworks or dependencies. It serves as a central hub to 
document and showcase IT projects as they are built and completed — 
linked directly from my LinkedIn profile and resume.

## Projects Showcased
- Cybersecurity — Home lab environments, SIEM, threat analysis
- AI / ML — Log anomaly detection, RAG chatbots, automation
- Agile / Delivery — Sprint simulations, Scrum documentation
- Development — Python scripting, REST APIs, automation suites
- Cloud — AWS deployments, Docker, CI/CD pipelines

## Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- HTML Canvas API (hero animation)
- Google Fonts — Jost, Cormorant Garamond
- Hosted on GitHub Pages (free)
